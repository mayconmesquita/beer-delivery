# Beer Delivery - A React Native delivery app

# Install dependencies

`npm install`

# Install on android

`npm run android `

# Run project

`npm start`

# Run tests:

`npm test`

# Download on the Google Play

https://play.google.com/store/apps/details?id=app.beerdelivery

PS: This project was only tested on Android.
